# Gold Price System

Hệ thống hỗ trợ theo dõi và dự báo giá vàng thế giới, tích hợp chatbot và kiến trúc microservices.

## Mục tiêu

- Theo dõi giá vàng thế giới
- Dự báo giá đóng cửa ngày kế tiếp
- Tích hợp chatbot hỏi đáp dựa trên dữ liệu hệ thống
- Gửi thông báo qua Discord

## Công nghệ chính

- Python
- Pandas
- Prophet
- FastAPI
- PostgreSQL
- React / Next.js
